# Farm SiK-Ness

**Farm SiK-Ness** es un videojuego multiplataforma (PC y Smartphone) desarrollado en Godot 4.5.1.

## Descripción
El juego combina mecánicas de **Idle** con gestión y granjeo. Los jugadores podrán cultivar elementos, procesarlos y venderlos, además de gestionar edificios en una cuadrícula con mecánicas de posicionamiento estratégico.

## Estructura del Proyecto
- `project/`: Contiene el proyecto de Godot.
- `docs/`: Documentación del proyecto.
  - `GDD/`: Documento de Diseño del Juego.
  - `mechanics/`: Definición de mecánicas.
  - `algorithms/`: Algoritmos y lógica del juego.
  - `diagrams/`: Diagramas de flujo y arquitectura.

## Tecnologías
- **Motor**: Godot Engine 4.5.1
- **Lenguaje**: GDScript
- **Plataformas**: Windows, Android (por definir)
